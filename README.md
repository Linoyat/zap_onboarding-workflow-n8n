## 🛠️ מה הולך כאן מבחינה טכנית?

### 🔎 איסוף נתונים (Scraping)

השתמשתי ב-Firecrawl כדי לסרוק את המיניסייט של הלקוח בדפי זהב ואת אתר הבית שלו - ביצעתי סריקה עמוקה של כל תתי הדפים באתר הבית כדי להבטיח איסוף נתונים מקסימלי.

---

### 🧹 ניקוי ועיבוד (The Code Node)

כתבתי קוד JavaScript ייעודי שמנקה את ה"רעש" מהדאטא הגולמי, שולף באופן אוטומטי מספרי טלפון וכתובות מייל, ומכין את כל המידע לסוכן ה-AI בצורה מובנית ומסודרת.

---

### 🧠 המוח (AI Agent)

סוכן AI המבוסס על מודל GPT-4 מנתח את הנתונים המעובדים. הוא לא רק "מעתיק" מידע, אלא מזקק תובנות:

* זיהוי ה-Unique Selling Point (USP) של העסק.
* הבנת שירותי הליבה של הלקוח.
* ניסוח תסריט שיחה אישי מהמפיק ללקוח שמותאם בדיוק לנתונים שנמצאו.

---

### 🖇️ אינטגרציה ל-CRM

בסוף התהליך, כל המידע נשלח ישירות ל-HubSpot המדמה את מערכת ה-CRM הארגונית. המפיק מקבל כרטיס לקוח מעודכן הכולל:

* פרטי קשר.
* תחומי התמחות ספציפיים.
* תסריט שיחה מוכן לשימוש מידי.

## 📝 סיכום ומה הלאה

זה אב-טיפוס עובד, אבל תמיד יש המון מקום לשיפור. מכיוון שהמטלה בוצעה בין הזמנים של החג - אם היה לי עוד קצת זמן הייתי מתמקדת ב:

* ייעול ניקוי הנתונים: שיפור עיבוד הטקסט לפני השליחה ל-Agent כדי לחסוך ב-Tokens ולשפר מהירות.

* אוטומציית הודעות: חיבור Node לשליחת הודעת וואטסאפ או טלגרם אוטומטית ללקוח ברגע שכרטיס הלקוח נוצר.

בסופו של דבר, המטרה היא לקצר תהליכים ולתת למפיקים של זאפ כלים חכמים יותר. מקווה שתהנו מהתוצר! 🦾
